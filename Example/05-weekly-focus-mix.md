<!-- Marker: How To Use -->
## How To Use

- Click any `#pomodoro` task in the plugin view to start a session.
- Subtasks under that task show up in the timer panel so you can tick them off while focusing.
- Markers like `<!-- Marker: Weekend -->` appear in the marker widget for quick navigation.
