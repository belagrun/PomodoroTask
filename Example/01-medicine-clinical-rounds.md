## Pomodoro Tasks

- [ ] Pre-round chart review for internal medicine service #pomodoro üçÖ:: 1/3
  - [ ] Identify top 3 active problems for each patient
  - [ ] Review vitals trends (24h) and oxygen requirements
  - [ ] Check morning labs and critical values
  - [ ] Reconcile active medications (renal dosing, interactions)
  - [ ] Confirm pending imaging and consult notes
  - [ ] Write one-liner summary for each patient
  - [ ] Prepare questions for attending (diagnosis, next steps)
  - [ ] Flag patients with discharge barriers
  - [ ] Note antimicrobial start/stop dates
  - [ ] Update problem list in a consistent order

- [ ] Antibiotic stewardship mini-audit (5 cases) #pomodoro üçÖ:: 2/5
  - [ ] Select 5 inpatient cases on antibiotics
  - [ ] Verify indication matches documented diagnosis
  - [ ] Check cultures (blood/urine/sputum) and susceptibilities
  - [ ] Review duration and stop dates
  - [ ] Assess IV-to-PO switch criteria
  - [ ] Identify opportunities for de-escalation
  - [ ] Confirm renal/hepatic dose adjustments
  - [ ] Document recommendations (short, actionable)
  - [ ] Prepare 2 teaching points for rounds
  - [ ] Communicate changes to the team (SBAR)
  - [ ] Re-check allergies and reaction details
  - [ ] Add follow-up reminder for 48‚Äì72h reassessment

- [ ] Draft 2 discharge summaries (high quality, concise) #pomodoro üçÖ:: 3/4
  - [ ] Confirm primary diagnosis and key secondary diagnoses
  - [ ] Summarize hospital course by problem
  - [ ] Reconcile discharge medications (start/stop/continue)
  - [ ] Add follow-up appointments and pending studies
  - [ ] Write patient-friendly instructions (red flags)

## Notes

- Keep the *one-liner* consistent: age/sex, PMH, reason for admission, current status.
- If a task contains a goal like `üçÖ:: 2/5`, the plugin will track progress toward 5 cycles.
